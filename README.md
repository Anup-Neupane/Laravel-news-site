# Laravel Vue and Inertia News Site

This is a simple News/Bloging Website Build Using Vue and Inertia with Laravel. 


## Quick Start
1. git clone https://github.com/tauseedzaman/Laravel-news-site.git
2. cd Laravel-news-site
3. cp .env.example .env
4. composer install
5. php artisan key:generate
6. php artisan migrate:fresh --seed
7. npm install && npm run dev
8. php artisan serve
9. [Open https://127.0.0.1:8000](http://127.0.0.1:8000).
